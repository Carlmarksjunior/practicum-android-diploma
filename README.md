# VacancySearcher 🔍

**Приложение для поиска вакансий** с использованием демонстрационного API.  
Удобный инструмент для мониторинга рынка труда с возможностью сохранять понравившиеся вакансии в локальную базу данных.

## ✨ Особенности

- **Бесконечная прокрутка** с фоновой загрузкой данных для плавного листания вакансий
- **Автоповтор запросов** при ошибках сети во время пагинации
- **Гибкая фильтрация**:
  - по месту работы
  - по отрасли
  - по ожидаемой зарплате
- **Сохранение избранных вакансий** в локальную базу данных приложения

## 🛠 Технологии и архитектура

### 🧱 Архитектура
- **Паттерн:** MVVM (Model-View-ViewModel)
- **DI (Внедрение зависимостей):** Koin

### 🧭 Навигация и интерфейс
- **Навигация:** Jetpack Navigation
- **Подход:** Single Activity + несколько Fragments
- **Безопасный доступ к UI:** ViewBinding

### 📱 Слой представления
- **ViewModel** и **LiveData** — для управления состоянием
- **Kotlin Coroutines** и **Coroutines Flow** — для асинхронной работы

### 🌐 Работа с API
- **Сетевые запросы:** Retrofit2
- **Сериализация JSON:** Gson
- **Загрузка и кэширование изображений:** Glide

### 💾 Хранение данных
- **Избранные вакансии:** Room (локальная БД)
- **Настройки фильтрации:** SharedPreferences

### 🔍 Инструменты анализа кода
- **Detekt** — инструмент статического анализа, запускается автоматически при создании Pull Request
